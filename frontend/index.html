🚀 빠른 시작

  1. 좌측 PDF 업로드 버튼 또는 뷰어의 드래그앤드롭 영역에 PDF를 올립니다. (여러 개·압축파일도 가능)
  2. 💡 파서 추천에서 현재 문서에 적합한 파서를 점수와 함께 추천받습니다.
  3. 문서 뷰어에서 파서를 선택하고 범위(현재페이지/전체/구간)를 정합니다.
  4. ▶ 분석을 클릭하면 진행 상황이 하단 상태바에 표시됩니다. (중지 가능)
  5. 우측 패널에서 텍스트·마크다운·요소목록·미디어·JSON 탭으로 결과를 확인합니다.

🖱️ PDF ↔ 결과 연동

  • PDF의 오버레이 박스를 클릭하면 요소 목록에서 해당 요소가 강조됩니다.
  • 요소 목록의 항목을 클릭하면 PDF의 해당 위치로 스크롤·강조됩니다.
  • 여러 페이지 PDF는 연속 스크롤로 이어보며, 높이맞춤/폭맞춤으로 보기를 조절합니다.

💡 파서 추천 분석 (객관적 방법론)

업로드한 문서를 측면별로 가장 신뢰도 높은 도구로 측정합니다:

  • 이미지: PyMuPDF get_image_info (정확한 임베드 이미지 감지)
  • 표·차트: 벡터 그래픽(선/곡선) 패턴 분석
  • 언어: langdetect + CJK/아랍 스크립트 분석
  • 레이아웃: 폰트 수·단(컬럼) 추정

빠른 분석(<1초)과 심층 분석(스캔 문서 샘플 OCR)을 선택할 수 있고, 각 측정의 도구·신뢰도·한계를 투명하게 공개합니다.

📊 파서 분석결과 (실측)

파서 정보 메뉴 상단에 샘플문서 실측 기반 품질 × 처리속도 산점도(오른쪽 위일수록 빠르고 정확)와, 파서별 기능·실측 성공률을 담은 비교표가 있습니다. "실측" 배지를 누르면 파서별 개조식 상세의견도 볼 수 있습니다.

🖼️ 멀티미디어 추출

그림·표·차트·수식을 잘라내어 미디어 탭에 썸네일로 모으고, 클릭하면 팝업으로 크게 봅니다. 마크다운에도 이미지가 함께 표시됩니다.

⚙️ 파서 옵션 & LLM

  • 각 파서의 ⚙ 옵션 버튼으로 상세 설정(언어, GPU/CPU, DPI 등)을 조절합니다.
  • GPU 지원 파서는 auto/CPU/CUDA(윈도우)/MPS(맥)를 선택할 수 있습니다.
  • LLM 파서(GPT/Claude/Gemini)는 인증 상태와 모델 선택이 파서 선택 옆에 표시됩니다.

📦 파서 설치

미설치 파서는 ⚠ 표시됩니다. 가상환경에서 설치하세요:

# 경량 파서
pip install pdfplumber pdfminer.six pypdf pymupdf4llm markitdown
# 고급 파서 (Python 3.10+)
pip install docling marker-pdf mineru
# OCR 파서
pip install easyocr paddlepaddle paddleocr ocrmypdf
# VLM (API)
pip install openai anthropic google-generativeai

🛠️ 시스템 의존성 (OCR용)

OCRmyPDF·Unstructured 등은 외부 도구가 필요합니다:

# macOS (Homebrew)
brew install poppler tesseract tesseract-lang ghostscript
# Windows: 각 도구 설치 후 PATH 등록

💻 실행 파일 만들기 (Mac/Windows)

PyInstaller로 Python 없이 실행되는 단독 실행 파일을 만들 수 있습니다:

pip install pyinstaller
pyinstaller --onefile --windowed run.py
# dist/ 폴더에 실행 파일 생성
